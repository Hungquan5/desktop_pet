# Changelog

## 1.1.0 - 2026-07-19

- Added a cozy edge-anchored desktop habitat with collapse, bottom-edge movement,
  click-through-aware masking, and deterministic cushion/snack/ball/box physics.
- Added a unified Home, Chat, Play, and Settings panel, quick pet actions,
  coachmark, reduced motion, muted-by-default original sound, and shared theme.
- Added original provenance-documented Momo v2 pixel-chibi animation assets with
  expressive held/eat/play/sleep/box/think/listen/talk roles.
- Added schema-3 atomic pet+habitat persistence, schema-2 companion settings,
  v1/v2/v3 character compatibility, idempotent rewards, and safe restoration.
- Added direct SmolLM habitat intents and candidate-constrained SmolVLM choices
  based only on a synthetic internal habitat scene, without desktop capture.

## 1.0.0 - 2026-07-19

- Added inspectable episodic/profile/task/relationship/procedural memory with FTS,
  deduplication, retention, management, backup, restore, export, and deletion.
- Added permission-gated timers, focus sessions, todos, reminders, notes,
  clipboard summaries, selected-file tools, application opening, and redacted audit.
- Added push-to-talk audio states, ephemeral capture, lazy Whisper in the existing
  worker, queued Qt TTS, interruption, echo suppression, and text fallback.
- Added privacy-first desktop metadata probes, deny lists, privacy mode, quiet
  hours, rate-limited proactive reasons, and coding/focus reactions.
- Added positive-only XP, affection, inventory, achievements, daily activities,
  toys, snacks, and a bounded mini-game.
- Added v2 persona/voice character packs, an original CC0 sample character,
  signed and integrity-checked plugins, bubblewrap isolation, quotas, and MCP stdio.
- Added onboarding, tray/settings/memory/task/plugin/audit controls, signed update
  manifests, release channels, migration backups, and expanded diagnostics.
