# Momo v2 asset provenance

- Created: 2026-07-19 for the VLA Pet project.
- Mode: OpenAI image generation, followed by deterministic chroma-key removal,
  equal-cell atlas cropping, and nearest-neighbor scaling.
- Source prompt: an original cozy pixel-chibi calf-girl desktop mascot in a
  4×4 animation atlas, with cream hair, cocoa horns/boots, tomato-red cape,
  and a flat `#00FF00` background. The prompt explicitly excluded existing
  characters, franchises, text, logos, signatures, and watermarks.
- Generated source: `source/momo_v2_atlas_chroma.png`.
- Transparent source: `source/momo_v2_atlas.png`.
- Runtime frames: `frames/frame_00.png` through `frames/frame_15.png`.
- Project license: CC BY 4.0, attributed to the VLA Pet project with OpenAI
  image generation.

The legacy prototype sprites are retained only as source-history material and
are not referenced by the v1.1 runtime or release manifest.
